## Campaign Manager

## Dependancies

```bash
npm install
# or
yarn install
```

## Getting started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Code Coverage

```bash
npm run test
npm run test:watch
# or
yarn test
yarn test:watch
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start managing your campaign.
